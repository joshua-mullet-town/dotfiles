# dotfiles

Personal configuration files synced across machines.

## iTerm2

iTerm2 preferences are automatically saved to `iterm2/` directory.

### Setup on a new machine:
1. Clone this repo
2. Open iTerm2 → Preferences → General → Preferences
3. Check "Load preferences from a custom folder or URL"
4. Browse to `/path/to/dotfiles/iterm2/`
5. iTerm2 will automatically sync your themes, profiles, and settings
